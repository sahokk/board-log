# BoardLog ペルソナ分析レポート

**作成日:** 2026-03-10
**目的:** ユーザー理解を深め、機能開発の優先順位を決定する

---

## 📋 エグゼクティブサマリー

BoardLogの「思い出アルバム」コンセプトを軸に、10種類のペルソナを設定し、各ユーザー層のニーズ・課題・離脱リスクを分析。**写真添付機能**、**プレイヤー記録機能**、**クイック記録モード**の3機能が最優先で実装すべきと結論。これらによりライトユーザーの定着率を大幅に向上できる。

---

## 👥 ペルソナ一覧

### 1. 📸 記録好きライトゲーマー「美咲（28歳・会社員）」

**プロフィール:**
- 月2-3回、友人とカジュアルにボドゲを楽しむ
- Instagram好きで写真を撮るのが日課
- BGGは英語で複雑すぎて挫折経験あり

**なぜ使い始めた？**
- 「どのゲームをいつ遊んだか」を写真のように残したかった
- BGGは英語で複雑すぎた
- 箱画像が可愛くて、見返すのが楽しそうだった

**なぜ使わなくなる？**
- 記録が面倒（毎回ゲーム検索→記録の手間）
- 友達に見せられない（公開機能なし）
- Instagramに投稿する方が反応があって楽しい

**なぜ使い続ける？**
- 自分だけの思い出帳として
- 「あのゲーム、前いつやったっけ？」がすぐ分かる
- ボドゲ名刺が可愛くてシェアできる

**求めていること:**
- 素早い記録（検索→記録を2タップで）
- 写真も一緒に保存したい
- 友達との思い出を共有したい

**あったら良い機能:**
- ✨ **写真添付機能**（プレイ風景や盤面）
- 🔗 **友達との共同記録**（一緒に遊んだ人をタグ付け）
- 📱 **インスタストーリー連携**（名刺を直接投稿）
- 🎲 **QRコード読み取り**（ゲーム検索の手間削減）

**使っていない機能:**
- 詳細な統計（興味なし）
- 評価システム（つけるの面倒）

**求めていること:**

## 📊 横断分析：機能要望ランキング

### 🔴 最優先（6名以上が要望）

#### 1. 📷 **写真・動画添付** - 6/10人
**要望者:** 美咲、健太、優奈、さくら、麻衣、(陽子)
**重要度:** ★★★★★

**分析:**
- 「思い出アルバム」コンセプトの核心
- プレイ風景・盤面写真は記憶を呼び起こす最強のトリガー
- 現状はテキストのみで「アルバム感」が弱い
- ライトユーザー層の定着率に直結

**技術実装:**
- Supabase Storage活用（既にプロフィール画像で実装済み）
- PlayRecordに画像URL配列を追加
- サムネイル生成で履歴一覧での表示最適化

---

#### 2. 👥 **一緒に遊んだ人の記録** - 5/10人
**要望者:** 美咲、健太、優奈、隼人、麻衣
**重要度:** ★★★★★

**分析:**
- ボドゲは本質的に社交的活動
- 「誰と遊んだか」は「いつ・何を遊んだか」と同じくらい重要
- 友達タグで検索できると「あの人と何遊んだっけ？」が解決
- 将来的なソーシャル機能の基盤

**技術実装:**
```typescript
model PlayRecord {
  // ...existing
  playerNames: String[]  // フリーテキスト配列
}
```
